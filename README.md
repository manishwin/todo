# Todo-list app

Simple doto app in which workw with api and html and javascript.


### `#Prerequisites:`

   `Install Node.js`

   `install Express`
   

## Here are the steps to start a project:
 `Git clone`

 `cd app`

 `npm init`

 `npm run start`

Now, you should be able to see the project running at http://localhost:3000 in your web browser.`
